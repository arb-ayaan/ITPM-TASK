# ITPM-TASK
A clean, minimalist, and lightweight To-Do Application landing interface built strictly using HTML. This project focuses on a "no-CSS-file" architecture, utilizing classic HTML attributes and inline styling to achieve a perfectly centered and responsive layout.

🚀 Features
Pure HTML: Built without external CSS frameworks or JavaScript.

Midpoint Alignment: The login and sign-up cards are perfectly centered horizontally and vertically using table-based structures.

User-Friendly Forms: Includes thick, accessible input fields for better UI/UX.

Navigation: Seamless linking between the Login and Sign-Up pages.

Responsive-Ready: Designed to stay centered across various screen sizes.

📸 Preview
The application consists of two main pages:

Sign-Up Page: Includes Username, Email, Password, and Confirm Password fields.

Login Page: Simple interface for existing users to access their tasks.

🛠️ Technology Used
HTML5: Structure and Semantic elements.

Inline Styles: Used for spacing (padding) and "thick" input field effects.

Table Layouts: Used for precise vertical and horizontal centering without flexbox/grid dependencies.
